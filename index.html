This language looks a little like a C based language but is actually pretty different.  This language is also not very efficent.  (It is made with JavaScript)


Here is some example code for printing Hello World! to the console to get you started:

=> "Hello World!";


What does this mean?

○ First off => takes all strings after it and prints them in output.

○ "Hello World!" is the string that is outputted by =>.

○ ; ends the line just like in most C based languages.

○ The program ends when there are no more characters left.
